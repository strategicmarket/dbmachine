---------------
> create a web action 
wsk action create create-cat create-cat.js --web true

Using the --web flag with a value of true or yes allows an Action to be accessible via REST interface without the need for credentials. 

A web Action can be invoked by using a URL that is structured as follows: https://{APIHOST}/api/v1/web/{namespace}/{packageName}/{actionName}.{EXT}.

https://console.bluemix.net/docs/openwhisk/openwhisk_webactions.html#openwhisk_webactions

----------------
invoke the action
wsk action invoke \
  --blocking \
  --param name Tahoma \
  --param color Tabby \
  create-cat

----------------
